# Phase 3 Plan: Offline Mode — Ollama First
# Kairo Phantom v3.0

## Objective
Flip Kairo's default from cloud to local. A user with Ollama installed and zero API keys must be able to use Kairo fully offline, immediately, with a one-liner setup prompt if Ollama is missing.

## Philosophy
"Offline" doesn't mean "worse." `qwen2.5-coder:14b` running locally is faster than a cloud round-trip for short document edits. This is a critical differentiator for enterprise and privacy-conscious users. We make offline the default; cloud is the power-user upgrade.

### 2. `ai.rs` — Refactor `OllamaBackend` for Production Use
Current `OllamaBackend` needs improvements:
- Use `/api/chat` endpoint (not `/api/generate`) for proper system prompt support
- Implement real streaming via the Ollama streaming API (line-delimited JSON, not SSE)
- Handle `"done": true` termination in the stream
- Proper context length handling
- 30-second timeout (Ollama can be slow on first load)

```rust
// Ollama /api/chat request format
{
  "model": "qwen2.5-coder:14b",
  "stream": true,
  "messages": [
    {"role": "system", "content": "<system_prompt>"},
    {"role": "user", "content": "<user_prompt>"}
  ]
}

// Ollama streaming response (line-delimited JSON, NOT SSE)
{"message": {"content": "Hello"}, "done": false}
{"message": {"content": " world"}, "done": false}
{"done": true}
```

### 3. `main.rs` — Startup Health Check

```rust
async fn check_ollama_health(base_url: &str) -> bool {
    let client = reqwest::Client::builder()
        .timeout(Duration::from_secs(5))
        .build()
        .unwrap_or_default();
    client.get(&format!("{}/api/tags", base_url))
        .send().await
        .map(|r| r.status().is_success())
        .unwrap_or(false)
}

// In main():
if config.model.provider == "ollama" {
    let base_url = config.model.base_url.as_deref().unwrap_or("http://localhost:11434");

        println!("⚠  Ollama not detected. Get offline AI in 2 commands:");

        println!("   brew install ollama            (macOS)");

        println!("   Or add an API key to ~/.kairo-phantom/config.toml for cloud mode.");
        println!("   Kairo will retry Ollama every 30s...");

        // Switch to fallback provider if available

        info!("✅ Ollama running at {}. Offline mode active.", base_url);
```
